# Flask

:::info
Flask 是一个轻量级的 WSGI Web 应用程序框架。
它的设计目的是使入门变得快速和简单，并能够扩展到复杂的应用程序。
它最初是围绕 Werkzeug 和 Jinja 的简单包装器，现已成为最受欢迎的 Python Web 应用程序框架之一。

:::
